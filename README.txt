dashbord:
	- http://localhost:3000/
	- list the products

Menu:  
	- Shopping List: Same us Home
	- User List: list the users and allows to add new users if admin
	- Register: allows normal user to register
	- login
	- logout

end points
	- /: home
	- /users: list of users
	- /register: normal users registration
	- /addUser: allows admin to register other admins or normal users
	- /addProduct: allows admins to add new product
	- /save_user: allows GUI to send user data for adding to the db
	- /save_product: allows GUI to send product data for adding to the db
	- /login: request login page
	- /logout: logout from the app
	- /validate_login: validate user information in the login form
